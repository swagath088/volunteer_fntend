# Volunteer Management System Frontend

A React-based frontend for managing volunteers.

## Features

* Volunteer Registration
* Volunteer Login
* Volunteer Dashboard
* Admin Delete Volunteers
* Show/Hide Password
* Loading Indicator
* Responsive UI

## Tech Stack

* React.js
* Axios
* React Router DOM
* CSS

## Backend API

https://volunteer-bckend.onrender.com

## Installation

npm install

npm run dev
